## To try it

- Git the repository

```sh
git clone https://github.com/flingyp/vadmire-admin.git
```

- Installation dependencies

```sh
cd vadmire-admin

pnpm install
```

- Run

```sh
pnpm run dev
```

- Build

```sh
pnpm run build 

pnpm run build:mock # mock mode
```
